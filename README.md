> Use [DO2-Momo/do2dk](https://github.com/DO2-Momo/do2dk) for a building do2iso images

<br/>

### Development 

Install build dependencies

```
pacman -Sy rust gtk4
```

Install runtime dependencies

```
pacman -Sy arch-install-scripts
```

<br/>

Build with

```
cargo run
```
